# 📚 QueryPDF – AI-Powered Document Assistant

QueryPDF is an AI-powered web application that allows users to:

* 📄 Scrape book data and store it as PDFs
* 🔍 Browse a catalog of documents
* 👁️ View PDFs inside the web app
* 🤖 Ask questions using AI (RAG + Groq LLM)

## 🚀 Features

* Web scraping (Books to PDF)
* FAISS vector database
* Retrieval-Augmented Generation (RAG)
* Groq LLM integration
* Clean UI with Flask + Tailwind

## 🛠️ Tech Stack

* Python, Flask
* LangChain
* Groq API (LLaMA models)
* FAISS
* HuggingFace Embeddings
* Tailwind CSS

## ⚙️ Setup

```bash
git clone <repo-url>
cd project
pip install -r requirements.txt
```

Create `.env`:

```
GROQ_API_KEY=your_key_here
```

Run:

```bash
python app.py
```
